**Problem Statement:**
Design a movie ticket booking system similar to BookMyShow that allows 
users to browse movies, select theaters and showtimes, book tickets, and manage reservations.


Requirements:
1. Users can search for movies/events by title
2. Users can browse movies playing at a given theater or venue.
3. Theaters have multiple screens.
4. Users can view available seats for a showtime and select specific ones
5. Users can book multiple seats in a single reservation; booking returns a confirmation ID
6. There can be different pricing tiers for seats/ add-ons (popcorn, colddrinks) 
7. Concurrent booking of the same seat: exactly one succeeds. No double booking
8. Users can cancel a reservation by confirmation ID, releasing the seats

Out of Scope:
- Payment processing (assume payment always succeeds)
- UI / rendering
